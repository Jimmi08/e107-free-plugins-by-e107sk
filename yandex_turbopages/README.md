# Yandex Turbo Pages RSS feed for CMS e107

## Description
The plugin adds RSS feeds for news and for selected pages

## Requirements
- e107 (>= v2.1.9)
- PHP 7 (tested with 7.0 and 7.2, but it should work with php 5.6 too )
